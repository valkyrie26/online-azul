Azul: A Digital Board Game

Pankhuri Shukla
ps4708@rit.edu

Motivation:

The goal was to bring my favourite board game, Azul, to life, using a beautiful and interactive web-based experience. I wanted to make something that was inspired by classic and limited edition versions of the game, like stained glass, chocolate and classic azul

How I did it:

Used Three.js to render a 3D board and tiles. User interaction was powered by mouse event listeners and physics logic. Textures, lighting, as well as calming water backgrounds were carefully tuned to create a fun experience!
1. Game board set-up, a flat box object added to scene with an image texture I found online (with increased contrast)
2. Orbital controls added to move tiles on top of the board
3. Factories around the board (cylindrical objects) with 4 randomly chosen tiles on each (box object)
4. Water background added to move 1/60th per second to appear animated
5. Ambient lighting added, light also added to the cursor that disables when we touch-drag a tile, and a point light inside the tiles to give them an extra "pop"
6. Collision mechanics added so the tile does not fall through the board or factories if dropped, and so that tiles do not spawn on top of each other (no overlapping allowed)
7. Tile textures - mosaic stained-glass (triangulated dual toned transparent), chocolate (bezier curved at edges, perlin noise for speckles), standard tiles with image texture

Future Work:

1. Ability to drop tiles into water with a splash
2. Multiplayer gameplay using socket programming
3. Custom soundtrack
4. Able to "snap" tiles into the correct box on the game board, certain amount of gravity
